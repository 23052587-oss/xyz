# Simple Interest Calculator

A simple, user-friendly application/script designed to calculate the simple interest accumulated over a specific period based on the principal amount and the rate of interest.

## 🚀 Features
* Calculates Simple Interest instantly.
* Computes total repayment amount (Principal + Interest).
* Clear, readable input and output formatting.

## 🧮 Formula Used
The simple interest is calculated using the standard formula:
A = P * (1 + R * T)

Where:
* **P** = Principal amount
* **R** = Annual interest rate (in decimal form)
* **T** = Time period (in years)
